---
title: Admin
description: Administrative routes for managing actors, keys, quotas, and STAR ledger operations.
---

These are platform administration routes restricted to users with admin scopes — they are not available to regular users or external agents.

Admin routes require a session with admin scopes (`admin:metrics` or `admin:debug`).

## GET /admin/actors

List all actors and their associated API keys.

**Auth:** Session | **Scope:** `admin:metrics`

---

## GET /admin/actors/\{actorId\}/quota

Quota counters for a specific actor.

**Auth:** Session | **Scope:** `admin:metrics`

---

## GET /admin/actors/keys/\{keyId\}

Admin view of any API key (not limited to owned keys).

**Auth:** Session | **Scope:** `admin:debug`

---

## PATCH /admin/actors/keys/\{keyId\}

Update mutable admin-managed fields on a key (e.g. rate limits, flags).

**Auth:** Session | **Scope:** `admin:debug`

---

## GET /admin/actors/keys/\{keyId\}/audit

Audit trail for any API key.

**Auth:** Session | **Scope:** `admin:debug`

---

## \$STAR Admin Routes

### GET /star/admin/supply

Global \$STAR supply report.

**Auth:** STAR internal token (`x-star-ledger-token`)

### GET /star/admin/transactions

Query transactions across all users.

**Auth:** STAR internal token

| Param | Description |
|-------|-------------|
| `from` | Start date (ISO) |
| `to` | End date (ISO) |
| `transactionType` | Filter by type |
| `initiatedBy` | Filter by user |
| `limit` | Max 500 |
| `offset` | Pagination offset |

### POST /star/admin/adjust

Administrative adjustment (reversal, correction, reclaim, grant).

**Auth:** STAR internal token + admin session

```json
{
  "idempotencyKey": "adj_001",
  "reason": "Duplicate reward correction",
  "adjustmentType": "correction",
  "postings": [
    { "accountId": "...", "amount": -50, "asset": "STAR" }
  ]
}
```
